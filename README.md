## Como rodar o projeto

<br />

Na raiz do projeto, rode o comando no terminal:

<br />

```shell
yarn install
```

<br />

Depois crie um arquivo chamado `.env`  e copie para ela tudo que está no arquivo `.develop.env`.

<br />

Rode o seguinte comando para rodar o sistema:

<br />

```shell
yarn start
```
